Based in Korea · 관심 분야: 펫테크, Database 취미, 뭐 소개 글 더 적기

맡은 일을 반드시 해내는 성실한 개발자,
끈기와 인내심을 겸비한 개발자

반려동물 장례 · 심리상담 · 추모 굿즈를 통합한 Last Page Project를 중심으로 웹 서비스 전체 구조를 설계하고 구현해 온 개발자입니다.

Spring Boot, JPA, MySQL을 기반으로 한 백엔드와 HTML/CSS/JavaScript, React를 활용한 프론트엔드를 함께 다루며, 실제 예약/결제/관리자 기능까지 연결되는 흐름을 경험했습니다.

새 기능을 빠르게 실험하고, 코드를 정리하며, 사용자 경험과 사용자의 시각을 우선시하여 개발하는 것에 중점을 둡니다.
